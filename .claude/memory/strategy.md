## Strategic Analysis

### Judging Weight Priorities
60% of score is technical (accuracy, reasoning, reliability). Build something that **works reliably** and demonstrates **clear multi-step reasoning** — these two criteria alone are 40% of the score.

### Prize Maximisation
- Best Overall Agent ($15k) can be won alongside a category prize. Building one strong Reasoning Agent could win both Best Reasoning Agent AND Best Overall Agent.
- Best Use of IQ Tools ($5k) is a separate award — deep/creative IQ integration gives two prize vectors from one project.
- Accessibility Award and Hack for Good are low-competition award vectors if the project fits.

### Track Recommendation (to validate with Tom)

1. 20% judging weight on "Reasoning & Multi-step Thinking" — directly aligned with the track theme
2. Microsoft Foundry is the most developer-accessible of the three toolsets for standalone builders
3. Strong overlap with Best Overall Agent eligibility
4. Foundry IQ satisfies the mandatory IQ requirement within the same ecosystem
